## **chip 4017**
El CD4017 es un circuito integrado digital que funciona como un contador decimal secuencial. Su principal característica es que activa sus salidas de manera ordenada, una por una, cada vez que recibe una señal de entrada.
contador decimal con 10 salidas (Q0 a Q9)
El cambio es secuencial y no simultáneo

 ¿Qué significa eso?
- Funciona como una compuerta lógica NAND
- Tiene mayor estabilidad frente a ruido
- Permite trabajar con señales más limpias

## **Chip 4093**
El 4093 es un chip que tiene 4 compuertas NAND especiales
Es como un interruptor inteligente 
Sirve para hacer señales más limpias (evita ruidos raros)
Muy usado para:
Osciladores (hacer que algo prenda y apague solo)
Temporizadores
Anti-rebote de botones
